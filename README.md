# Market Boi App

Market Boi is a local market shopping and delivery application that connects customers with trusted market agents.

## Features (Planned)
- Customers place market orders
- Agents negotiate prices on behalf of customers
- Delivery to customer location
- Real-time order updates

## Goal
To digitize local market shopping and make it easier, faster, and more transparent.

## Status
🚧 Project is under development.
